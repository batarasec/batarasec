# Scanner Upload Guide

> Draft placeholder for the public Community repository.

BataraSec Community accepts supported scanner outputs through the web UI or API.

Planned scanner examples:

- Trivy JSON
- OWASP ZAP JSON/XML
- Semgrep JSON
- Nmap XML
- Nikto JSON/text where supported
- Selected bug bounty scanner outputs where supported

Use only scanner output from systems you are authorized to test.
