# VM Agent Installation Guide

> Draft placeholder for the public Community repository.

The BataraSec VM agent can enroll a Linux host and send vulnerability and hardening findings to the platform.

Community v0.1 planned scope:

- Linux VM agent install.
- Enroll with a project token.
- Initial scan.
- Platform-triggered Scan Now.

Enterprise features such as advanced release rollout, fleet governance, and additional platform agents may require Enterprise.
