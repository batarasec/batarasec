# Contributing

BataraSec Community is not accepting public pull requests in the near term.

The public Community repository is planned as a distribution repository for installer, Docker Compose, documentation, and sample data. The application source code remains private/internal until a later source-opening decision.

## Feedback

Once public issues are enabled, users may report:

- Installation problems.
- Documentation gaps.
- Community image/runtime issues.
- Scanner upload compatibility issues.
- Feature requests for Community or Enterprise.

## Pull requests

Public pull requests are not accepted yet. This policy may change after the Community/Enterprise source boundary is finalized.

## Security issues

Do not open public issues for security vulnerabilities. See `SECURITY.md`.

## Edition boundaries

Please do not request or submit changes that bypass Community limits, Enterprise licensing, Portal/Central KB gating, or commercial feature controls.
